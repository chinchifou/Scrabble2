REQUIREMENTS

python  
pygame
(see : "help_for_install.txt" for further information)


SETTING UP THE GAME

TO change your peferences you can go to : "...\Scrabble2\config\"
- Edit "game_rules.ini" to add a new player or to create custom rules
- Edit "display_settings.ini" to change your display settings preference (default is fullscreen)


LAUNCH THE GAME

On Windows :
In "...\Scrabble2\sources\" double-click on "launcher.bat" or "main.py". The effect is the same.
You can create a windows shortcut to an of these files. You can freely use the icon provided in "...\Scrabble2\assets\icon\" to custom this link.


HOW TO PLAY

The classic rules of Scrabble are applied to calculate the score (it is possible to get several multiplier at once)
Invalid moves make the game crash ...
There is no other limitation (timer, automatic ending, ...)
To quit just press the key "ESCAPE"

Have fun !
